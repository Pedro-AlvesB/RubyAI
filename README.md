💎 RubyAI — Assistente Inteligente Web










RubyAI é uma aplicação web de assistente inteligente com interface moderna, integração com IA generativa, suporte a imagens, voz, clima em tempo real e personalização de tema.

O projeto foi desenvolvido com foco em experiência do usuário, organização modular e integração com APIs externas.

✨ Funcionalidades

💬 Chat com IA generativa

🖼 Upload e descrição de imagens

🎙 Entrada por voz (Speech Recognition)

🔊 Leitura de respostas (Text-to-Speech)

🌤 Widget de clima em tempo real

🎨 Sistema de temas personalizáveis

💾 Salvamento de conversas

📱 Interface responsiva

🔌 Estrutura preparada para integração com Arduino (controle de LED)
